

## 介绍

2022年暑假字节跳动青训营项目，模仿[掘金官方网站](https://juejin.cn/)。

主要制作网页前端部分，包括导航头，话题列表，话题详情和侧边导航栏等。

## 启动

1. 克隆项目到本地仓库。
```bash
git clone git@github.com:ChenYuTong10/juejin-official-website.git
```

2. 切换到项目根目录下。
```bash
cd juejin-official-website
```

3. 安装项目依赖。
```bash
npm install
```

4. 启动项目
```bash
npm run dev
```

## 技术栈

1. Vue3.x
2. Vite
3. Typescript
